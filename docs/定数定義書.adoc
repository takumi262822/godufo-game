== 概要

本書はプロジェクト「{project}」で利用する定数値を列挙したものです。
ソースファイル `src/utils/constants.js` から逆引きで抽出しています。

---

== GAME_CONFIG

[cols="2,1,1,4", options="header"]
|===
|定数名
|型
|値
|内容

|CANVAS_WIDTH
|number
|600
|Canvas の描画幅 (px)

|CANVAS_HEIGHT
|number
|750
|Canvas の描画高さ (px)

|START_SHIELD
|number
|100
|ゲーム開始時のシールド初期値

|MAX_POWER_LEVEL
|number
|5
|プレイヤーの最大パワーレベル

|SPECIAL_ENERGY_MAX
|number
|100
|特殊攻撃メーターの上限値

|===
