=== 各コードの意味

[cols="2,3", options="header"]
|===
|フィールド名
|説明

|name
|敵テンプレートの識別コード。EnemyManager ウェーブ生成時に使用

|shape
|Canvas 描画時の形状を示す内部コード

|hue
|色相値 (0〜360)。描画時の fillStyle に使用

|hp
|生成時の初期 HP。ダメージ計算の基準値

|speedMult
|基底スピードに乗じる倍率 (1.0 = 標準)

|agility
|方向転換ベクトルの量。大きいほど動きがランダム

|shootRate
|1フレームあたりの弾発射確率 (0〜1)

|score
|撃破時に加算されるスコア

|size.w / size.h
|Canvas 描画矩形の幅・高さ (px)

|===
