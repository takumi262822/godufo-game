== 概要

本書はプロジェクト「{project}」に登場するコード値（識別子・種別値）を列挙したものです。
ソースファイル src/data/enemy-data.js, src/utils/constants.js から逆引きで抽出しています。

|項番
|コード和名
|コード英名
|データ型
|桁数

|#0ff
|文字型
