== 概要

本書は宇宙大戦（GODUFO ゲーム）全体で使用する共通ユーティリティの仕様を記載します。 +
DOM 反映前は必ず XssSanitizer を経由し、入力値は FormValidator で検証してから利用する。

---

== 1. XssSanitizer クラス

=== 格納先

`src/utils/xss.js`

=== 説明

DOM への文字列反映前に HTML 特殊文字をエスケープし、XSS 攻撃を防ぐ。

=== 静的メソッド

==== escapeHtml(value)

[cols="2,2,4", options="header"]
|===
^.^|項目 ^.^|型 <.^|内容

<.^|引数 `value` |any |任意の値。`null` / `undefined` は空文字に変換してから処理する。
<.^|戻り値 |string |`& < > " '` を HTML エンティティに置換した文字列
|===

.変換表
[cols="1,1", options="header"]
|===
|入力文字|変換後
|`&`|`&amp;`
|`<`|`&lt;`
|`>`|`&gt;`
|`"`|`&quot;`
|`'`|`&#39;`
|===

NOTE: `XSSSanitizer` は後方互換エイリアスとして同ファイルで export されている。

---

==== sanitizeTextInput(value)

`escapeHtml` への委譲メソッド。テキスト入力を DOM に反映する直前に呼ぶ。

[cols="2,2,4", options="header"]
|===
^.^|項目 ^.^|型 <.^|内容

<.^|引数 `value` |any |任意の値。`null` / `undefined` の場合は空文字を返す。
<.^|戻り値 |string |エスケープ済み文字列（`escapeHtml` の結果と同一）
|===

---

==== isInt(value)

[cols="2,2,4", options="header"]
|===
^.^|項目 ^.^|型 <.^|内容

<.^|引数 `value` |any |検査対象の値
<.^|戻り値 |boolean |`Number` 変換後に `Number.isInteger` で確認した結果
|===

---

==== inRange(value, min, max)

[cols="2,2,4", options="header"]
|===
^.^|項目 ^.^|型 <.^|内容

<.^|引数 `value` |any |検査対象の値
<.^|引数 `min` |number |下限値（含む）
<.^|引数 `max` |number |上限値（含む）
<.^|戻り値 |boolean |NaN 除外後に `min <= n <= max` を確認した結果
|===

---

==== validateScore(score)

[cols="2,2,4", options="header"]
|===
^.^|項目 ^.^|型 <.^|内容

|===
