== クラス設計

=== Game（game-core.js）

[cols="2,1,4", options="header"]
|===
|メソッド
|可視性
|処理内容

|constructor()
|public
|Canvas 要素取得、各マネージャーのインスタンス化、InputManager 初期化

|gameLoop()
|public
|update() → draw() → requestAnimationFrame によるループ再帰

|update()
|public
|フレームカウント増加、プレイヤー更新、敵弾生成、弾更新、衝突解決、レベル更新、UI 更新。ゲームオーバー条件を検知した場合は isGameOver = true に設定

|draw()
|public
|RenderManager.drawScene() に委譲して全描画を実行

|fire()
|public
|ProjectileManager.spawnPlayerFire() を呼び出す。isStarted かつ !isGameOver の場合のみ実行

|triggerSpecial()
|public
|SpecialManager.triggerSpecial() を呼び出す。isStarted かつ !isGameOver の場合のみ実行

|getUfoStyle()
|public
|スコアに応じた色・ズーム・光輪数を返す。score >= 10000 → GOD モード、score >= 3000 → OVERDRIVE、それ未満 → 通常

|===

=== EnemyManager（enemy-manager.js）

[cols="2,1,4", options="header"]
|===
|メソッド
|可視性
|処理内容

|constructor(types)
|public
|ENEMY_TYPES を受け取り enemies/bullets 配列を初期化

|update(frame, timeScale, width)
|public
|フレーム数を元にウェーブタイミングを判定し spawnWave() を呼び出す。全敵の座標更新と弾生成を実行し、生成した弾配列を返す

|spawnWave(wave, width)
|private
|ENEMY_TYPES からランダム選択した敵を canvas.width 内に配置

|===

=== CollisionManager（collision-manager.js）

[cols="2,1,4", options="header"]
|===
|メソッド
|可視性
|処理内容

|resolve()
|public
|プレイヤー弾 × 敵、敵弾 × プレイヤーの AABB 判定を実行。ヒット時に HP 減算・score 加算・shake 付与を行う。ボス撃破時は shake に 60 を設定

|===

== ゲーム進行フロー

[source]
----
1. ページ読み込み → new Game().gameLoop()
2. スタートボタン押下 → isStarted = true
3. gameLoop() 実行
   a. update(): 敵生成・弾移動・衝突判定・レベル進行
   b. draw(): Canvas 全体の再描画
4. シールドが 0 以下 → isGameOver = true → ゲームオーバー画面
----
